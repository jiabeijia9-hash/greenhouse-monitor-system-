# 温室监控系统项目

基于 STC15W4K58S4 单片机的温室环境监控系统上位机软件。

## 功能特性

- 📊 实时显示温湿度、光照强度等环境参数
- 📈 数据图表可视化展示
- 🔧 支持串口通信配置
- 💾 数据记录与查询功能
- 🎨 现代化 Material Design 界面

## 技术栈

- C# (.NET Framework)
- ReaLTaiizor Material Design UI 库
- 串口通信协议

## 使用方法

直接运行 `温室监控系统项目.exe` 即可启动应用程序。

## 硬件支持

- STC15W4K58S4 单片机
- 温湿度传感器
- 光照传感器
